## Ejemplos de uso


- Listar tareas

> node app listar

- Crear tarea

> node app crear -d 'hacer las compras'

- Actualizar tarea como realizada (true)

> node app actualizar -d 'hacer las compras'

- Borrar una tarea (retorna 'Tarea eliminada')

> node app borrar -d 'hacer las compras'
